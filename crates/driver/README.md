# µcad driver

This crate contains the µcad driver API.
If you want to integrate µcad into your application, use this API like this:

```rust
use microcad_driver::prelude as mu; // Import all symbols with `mu` prefix
use mu::traits::*; // Use all API traits 
```

> Note: The driver API is a work in progress and subject to change.
