# distribute_grid

Distribute a 2D or 3D geometry on a grid on the Z plane.

## Arguments

- `x: Length`: X offset of the grid.
- `y: Length`: X offset of the grid.
- `width: Length`: Overall width of the grid.
- `height: Length`: Overall height of the grid.
- `rows: Integer`: Number of rows in the grid.
- `columns: Integer`: Number of columns in the grid.
