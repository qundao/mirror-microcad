# Workbench Types

*Workbenches* come in three flavors:

| Type                          | Keyword  | Input Model | Output Model |
| ----------------------------- | :------- | :---------: | :----------: |
| [*sketches*](sketches.md)     | `sketch` |   *none*    |      2D      |
| [*parts*](parts.md)           | `part`   |   *none*    |      3D      |
| [*operations*](operations.md) | `op`     |  2D or 3D   |   2D or 3D   |
